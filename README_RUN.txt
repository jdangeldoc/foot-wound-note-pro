OrthoCoder Pro — One-Click Voice Fix

WHAT THIS DOES
- Installs the correct OpenAI SDK (1.x) and supporting libs
- Replaces wound_voice\voice_io.py with a tested 1.x-compatible version
- Prompts for OPENAI_API_KEY if not present (for this session)
- Launches `streamlit run app.py`

HOW TO USE (Windows, from your project folder):
1. Click the download link
2. Save the ZIP file to the `Downloads` folder
3. Go to `C:\Users\{YourUsername}\Downloads`
4. Right-click the ZIP > Extract All
5. Move the extracted file(s) to the correct folder (e.g., OneDrive – Jefferygroup\DocProjectVault\orthocoder-pro)

Then:
- Open File Explorer to your project folder (the one containing app.py)
- Right-click in the white space → Open in Terminal
- Run:  one_click_voice_fix.bat
- Paste your OPENAI_API_KEY when prompted
